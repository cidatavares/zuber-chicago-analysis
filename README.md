# 🚖 Zuber Chicago Taxi Analysis

## 📌 Sobre o projeto

Projeto desenvolvido durante o Bootcamp de Data Analytics da TripleTen.

O objetivo foi analisar dados de corridas de táxi da cidade de Chicago para identificar padrões de comportamento dos passageiros e avaliar a influência das condições climáticas nas viagens.

---

## 📊 Objetivos

- Explorar os conjuntos de dados
- Limpar e validar os dados
- Analisar empresas de táxi
- Analisar bairros com maior número de corridas
- Investigar o impacto das condições climáticas
- Interpretar resultados estatísticos

---

## 🛠 Tecnologias

- Python
- Pandas
- SQL
- Matplotlib
- SciPy

---

## 📈 Principais análises

✔ Empresas de táxi com maior número de viagens

✔ Bairros mais populares

✔ Distribuição das corridas

✔ Relação entre clima e duração das viagens

✔ Testes estatísticos
